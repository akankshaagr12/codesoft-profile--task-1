# 🌐 Personal Portfolio Website

This project is developed as **Task 1** of the **CodSoft Web Development Internship**.

It is a responsive personal portfolio website created using **HTML** and **CSS** to showcase my profile, skills, projects, achievements, and contact information.

---

## 🚀 Features

- 🏠 Home Section
- 👩 About Me
- 💻 Skills Section
- 📂 Projects Showcase
- 🏆 Achievements Section
- 📄 Resume Download Option
- 📞 Contact Information
- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
